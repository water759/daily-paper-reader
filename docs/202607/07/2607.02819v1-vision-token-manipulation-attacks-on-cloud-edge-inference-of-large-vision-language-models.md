## 摘要
云边大视觉-语言模型（LVLM）推理通过在边缘设备与云服务器之间拆分计算来实现高效部署。在此过程中，中间视觉令牌通过通信链路从边缘传输到云端，从而暴露了新的攻击面。我们研究了黑盒中间人设置下的视觉令牌操纵攻击（VTM-Attack），其中攻击者在预算约束下拦截并操纵所传输视觉令牌的一个子集。我们提出了四种朴素攻击策略以及一种基于优化的令牌选择方法。在4个基准测试中对6个最先进的LVLM（3B-72B）进行的实验表明，仅操纵10%的视觉令牌即可将准确率降低高达88.31%。这些结果揭示了云边LVLM推理中的关键漏洞。

## Abstract
Cloud-edge Large Vision-Language Model (LVLM) inference enables efficient deployment by splitting computation between edge devices and cloud servers. In this process, intermediate vision tokens are transmitted from the edge to the cloud over a communication link, thereby exposing a new attack surface. We study vision token manipulation attack (VTM-Attack) under a black-box man-in-the-middle setting, where an adversary intercepts and manipulates a subset of transmitted vision tokens under a budget constraint. We propose four naïve attack strategies and an optimization-based token selection method. Experiments on 6 state-of-the-art LVLMs (3B-72B) across 4 benchmarks show that manipulating only 10\% of vision tokens can reduce accuracy by up to 88.31\%. These results reveal a critical vulnerability in cloud-edge LVLM inference.

## 2. 方法论
- **核心思想**：攻击者在边缘与云之间的通信链路上拦截视觉令牌，在预算约束下（仅能修改一定比例的令牌）替换为恶意版本，破坏模型生成结果。
- **朴素攻击策略**（四种）：
  - **置换（Permutation）**：随机对所选令牌进行非恒等排列，打乱位置顺序。
  - **掩码（Masking）**：将所选令牌置零，消除其信息。
  - **高斯扰动（Gaussian Perturbation）**：添加与令牌范数成比例的高斯噪声。
  - **符号翻转（Sign Flip）**：令 `v̂_i = -v_i`，保持范数不变但逆转方向。
- **优化令牌选择方法**：
  - **目标**：最大化自注意力破坏损失 `L_SA = ||Attn(Ṽ) - Attn(V)||_F^2`，同时引入范数感知正则化 `L_norm = Σ w_i * (||v_i||^2 / sum||v_j||^2)` 以抑制对高范数令牌的偏见。
  - **总目标**：`max_{w∈[0,1]^L_v} L = L_SA - λ L_norm`，通过投影梯度上升求解，经T步迭代后取Top-k权重对应的令牌进行符号翻转。
- **算法流程**：初始化权重 → 循环迭代（构建连续松弛令牌、计算梯度、更新权重、裁剪至[0,1]、缩放至预算约束）→ 得到离散索引集 → 应用符号翻转。

## 6. 主要结论与发现
- 云边LVLM推理对视觉令牌操纵高度脆弱，仅10%的令牌即可导致灾难性性能下降（最高88.31%准确率损失）。
- **符号翻转是最强攻击**，在Qwen模型上常使准确率降至接近零，而置换最弱。
- **优化选择显著增强攻击**：相比随机选择，基于自注意力破坏与范数正则化的方法进一步降低准确率（如Qwen2.5-VL-3B在OCRBench上从35.70%降至30.50%）。
- **模型家族间鲁棒性差异大**：Qwen2.5-VL系列远不如InternVL3.5系列稳健；InternVL3.5中较大模型更鲁棒，但Qwen无此规律。

- **攻击策略多样且实用**：提出四种朴素策略和优化选择方法，尤其是符号翻转+优化选择在低预算下高效。
