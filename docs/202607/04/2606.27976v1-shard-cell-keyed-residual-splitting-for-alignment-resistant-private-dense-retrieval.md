## 摘要
密集嵌入支撑着语义搜索和RAG，然而一个泄露的向量存储会将大量底层文本交还给任何持有它的人。使这种攻击成为可能（少样本对齐、零样本反演、无监督跨空间翻译）存在一个共同的弱点：受保护的存储是一个单一的全局几何结构，可以对齐到已知的几何结构。常见的轻量级防御——秘密全局旋转——也不例外：一旦攻击者拥有大约子空间维度的已知对，正交Procrustes就可以恢复它。我们提出了Shard，一种保留检索的嵌入变换，它移除了这个薄弱轴。中心化的嵌入被分割成一个短的公开前缀（用于第一阶段检索）和一个私有残差，该残差在独立的秘密密钥下被分割成C个单元；残差在CKKS下进行重排序，其中密钥相互抵消，内积保持不变。单一参数C使设计从它所替代的全局线性基线（C=1）运行到每文档微密钥（C=N）。由于重排序是全维度的，Shard返回了半SVD截断所放弃的原始空间nDCG@10；并且由于残差是按单元局部密钥化的，在扩散的已知明文泄露下将其映射回公共框架所需的锚点数量大约是C倍（中位数从200到102,400，当C=256时），仅需少量加密查询。短的公开前缀泄露的邻居结构要少得多，而微密钥极限通过不可链接、可更新的模板将残差图驱动至零。该屏障能够抵御学习的、非线性和无监督的对齐器，并且在匹配效用的噪声防御几乎去匿名化每个探测时，Shard没有去匿名化任何探测。我们坦诚地说明其局限性：在单元内密钥会抵消，针对性的攻击者只需要大约d_priv个锚点，并且重叠的参考语料库仍然通过前缀泄露信息。Shard是一种攻击感知的几何防御，而非密码学保证。

## Abstract
Dense embeddings underpin semantic search and RAG, yet a leaked vector store hands much of the underlying text back to whoever holds it. The attacks that make this possible (few-shot alignment, zero-shot inversion, unsupervised cross-space translation) share one weakness: the protected store is a single global geometry that can be aligned to a known one. A secret global rotation, the usual lightweight defence, is no exception: orthogonal Procrustes recovers it once the attacker has about the subspace dimension in known pairs.   We introduce Shard, a retrieval-preserving embedding transform that removes this weak axis. The centred embedding is split into a short public prefix (for stage-1 retrieval) and a private residual sharded into C cells under separate secret keys; the residual is reranked under CKKS, where the keys cancel and leave the inner product exact. A single parameter C runs the design from the global-linear baseline it replaces (C=1) to per-document micro-keys (C=N). Because the rerank is full-dimensional, Shard returns the raw-space nDCG@10 that half-SVD truncation gives up; and because the residual is keyed cell-locally, mapping it back to a common frame under a diffuse known-plaintext leak costs roughly C times more anchors (median 200 to 102,400 at C=256), for a few encrypted queries. The short public prefix leaks far less neighbour structure, and a micro-key limit drives the residual graph to zero with an unlinkable, renewable template. The barrier holds against learned, non-linear and unsupervised aligners, and where a matched-utility noise defence de-anonymises almost every probe, Shard de-anonymises none. We are plain about the limits: within a cell the keys cancel, a targeted attacker needs only about d_priv anchors, and an overlapping reference corpus still leaks through the prefix. Shard is an attack-aware geometric defence, not a cryptographic guarantee.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：密集嵌入向量（dense embeddings）泄露后，攻击者可通过少样本对齐、零样本反演、无监督跨空间翻译等现代攻击手段还原文本。现有的轻量级防御——单个秘密全局旋转——是可对齐的，因为正交Procrustes攻击只需约一半维度（d/2）的已知明文-密文对即可恢复旋转。
- **整体目标**：设计一种**保留检索质量**的嵌入变换，从根本上移除全局几何这一弱轴，使存储的嵌入向量对对齐攻击具有弹性。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将中心化嵌入分解为**短公共前缀**（用于粗粒度第一阶段检索）和**私有残差**，残差按公共前缀上的k-means聚类划分为C个单元（cell），每个单元使用独立的秘密正交密钥旋转残差。服务端仅存储前缀和密钥化残差；重排序时在CKKS下加密计算内积，因密钥正交性，内积精确等于原生残差内积（⟨Hc rq, Hc ri⟩ = ⟨rq, ri⟩）。一条参数C控制从全局线性基线（C=1）到每文档微密钥（C=N）的平滑过渡。
- **关键技术细节**：
  - 使用PCA旋转将嵌入分为d_pub维前缀u（前几个主成分）和d_priv维残差r。
  - 对每个cell c，生成密钥Hc ∈ O(d_priv)（Householder反射乘积）。
  - 存储表示：{u_i, z_i = H_{c(i)} r_i}。
  - 查询阶段：客户端对每个活跃cell发送Enc(Hc r_q)；服务端计算ct-pt点乘；密钥抵消得到精确内积，结合前缀分数重建全维度相似度。
  - 重排序是全维度的（无SVD截断损失），且只在K_cands个候选上进行。
- **公式/算法流程**：
  - 离线：编码→PCA→前缀/残差分裂→k-means分cell→每cell密钥化残差→存储。
  - 在线：编码→PCA→本地前缀ANN短列表→给每个活跃cell发送加密残差查询→服务端ct-pt重排序→解密排序。

## 6. 论文的主要结论与发现

- **效用**：SHARD的全维度重排序恢复了原始nDCG@10，显著优于半SVD截断（后者在BEIR上损失2–8点）。
- **对齐抗性**：
  - 扩散已知明文攻击所需锚点约增长C倍（基线200→C=64时25,600→C=256时102,400）。
  - 针对性攻击（锚点集中在目标cell）仍需约d_priv（约320–576）个锚点，独立于C。
  - 学习线性（ALGEN）、非线性MLP、无监督vec2vec攻击均未超越正交Procrustes（后者最优）。
- **公共前缀泄露**：泄露的邻居结构远少于基线（NN-overlap@10: 0.20–0.55 vs. 0.76）。
- **微密钥极限**：消除残差图泄露（恢复率为0.00），且不可链接（re-keying AUC ≈ 0.5）。
- **与DP噪声对比**：匹配效用下DP噪声去匿名化R@1≈1.0，SHARD为0.0。
- **局限性**：
  - 同单元内密钥抵消，服务器可计算同单元相似性（cell粒度泄漏）。
  - 重叠参考语料库仍可通过公共前缀去匿名化。
  - 非密码学保证，访问模式未保护。

## 7. 优点：方法与实验设计亮点

- **方法创新**：将生物识别中的取消模板（cancellable templates）与CKKS同态加密结合到文本检索场景，通过cell-keyed残差分裂解决全局几何可对齐的根本弱点。
- **参数化灵活性**：单一参数C调节从全局线性到每文档微密钥的连续谱，提供可操作的效率-安全性权衡。
